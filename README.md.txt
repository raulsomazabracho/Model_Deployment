# Machine Learning App: Titanic Competition

In this link you can find the deployed model: https://titanicreg.herokuapp.com/


Los siguientes archivos son necesarios para que la aplicación se ejecute correctamente en la nube a través de la plataforma Heroku:
* runtime.txt: Pyhton version
* Procfile: commands for cloud execution con comandos para la ejecución en cloud.
* requirements.txt: libraries to import to the cloud .
* setup.sh: console commands to deploy streamlit in the cloud
*Prediccion.ipnnb --> Jupyter Notebook with the machine learning algorithm
